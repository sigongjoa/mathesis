# Node 7: Error Note System - Product Requirements Document (PRD)

**Version**: 1.0.0
**Last Updated**: 2026-01-09
**Status**: Design Phase
**Owner**: Mathesis Platform Team

---

## 1. 개요

**Error Note System (Node 7)**은 선생님이 학생들의 오답을 메타인지 관점에서 분석하고, Anki 간격 반복 알고리즘을 통해 체계적으로 복습할 수 있도록 돕는 시스템입니다.

### 1.1 배경 및 필요성
- 기존의 오답노트는 단순히 문제를 다시 풀어보는 데 그쳐, 왜 틀렸는지에 대한 깊은 성찰(메타인지)이 부족함.
- 선생님이 여러 학생의 오답을 일일이 관리하고 최적의 복습 시점을 챙겨주기 어려움.

---

## 2. 사용자 페르소나

### 2.1 중등 수학 선생님 (박교사)
- **니즈**: 학생들의 오답 데이터를 체계적으로 축적하고, 학생 개개인의 취약점을 한눈에 파약하고 싶음.
- **Pain Points**: 오답노트 검사가 너무 오래 걸리고, 학생이 정말 이해했는지 확인하기 어려움.

---

## 3. 기능 요구사항

### 3.1 FR-1: 오답노트 생성 및 관리
- **FR-1.1**: 이미지 업로드 시 OCR을 통한 문제 자동 인식.
- **FR-1.2**: 메타인지 5단계 프레임워크(이해/실수지점/유형/방지/실천) 입력 지원.
- **FR-1.3**: 학생별 오답노트 보관함 제공.

### 3.2 FR-2: Anki 기반 복습 스케줄링
- **FR-2.1**: SM-2 알고리즘을 적용하여 다음 복습일 자동 계산.
- **FR-2.2**: 복습 누락 시 알림 발송.

### 3.3 FR-3: AI 변형 문제 생성
- **FR-3.1**: 원본 문제와 유사한 개념을 사용하는 변형 문제 자동 생성.
- **FR-3.2**: 난이도 조절 기능 제공.

---

## 4. 성공 지표 (KPI)
- 오답 정답률 개선도 (전후 비교)
- 학생의 메타인지 답변 충실도
- 선생님의 오답 관리 소요 시간 감소 (기존 대비 50% 이상)
